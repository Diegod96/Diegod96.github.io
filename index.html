Covid-19 Dashboard

A Covid-19 Dashboard web application home to three applications. The first is a Covid-19 Global Dashboard that displays charts and information for the virus globally and by country. This was built using React and Material UI. The second application is a United States Covid-19 Dashboard built using Svelte JS and Bulma CSS. Charts are derived via the Chart JS library, and Moment was used to handle this time series data. The third application is a Covid-19 Probability Detector to see, based off a user's symptom's, if they have contracted the virus. This was built using Python Flask framework, styled with Bootstrap CSS, and predictions were made by using a logistic regression model. The last application is a Handwashing application that times the CDC 20 second guideline for washing your hands as well as display the correct way to do so. Built using Svelte JS.

Automated Craigslist Scraper

This is my Craigslist scraper application. Built using Python and using the Selenium automated webdriver framework, this script scrapes listings on craigslist based on type of item, location, max price, etc. This application is deployed/hosted on the Heroku cloud and runs a scrape every hour. It then sends myself a timestamped email with a csv file of the listings that were scraped.
