Live Twitter Dashboard

This application is a twitter sentiemnt dashbaord. The user can view the live sentiment of twitter along with twitter's sentiment over the current week. The user can also see the live/historic sentiment for a specififc topic or term (i.e. covid, nba, etc.). The live portion utilizse SQLite for quick index retrievals while the historic pie chart utilizes Apache Spark/Pyspark, AWS Kinesis Firehose, AWS S3, and MySQL to handle the ETL pipeline for large amounts of data. The piechart is updated every thirty minutes after the completeion of a Databricks notebook job.

Team Management Application

A PWA that allows a user (a coach) to add rowers and their information (name, email, 2k time, etc.) to a list view. The user can then see a pie chart visualization of how the team is composed based on the boat (Varsity 8, JV 4+, etc.) the rowers are in. Currently implementing forgot password functionality along with the ability for new users to make accounts. In the meantime, use "user" and "password" as login credentials to use and view the application. Built using Java Spring Boot / Spring Security and the Vaadin framework. JUnit tests developed to test basic functionality along with an H2 DB & MySql DB were used for local development. Currently deployed on an AWS Elastic Beanstalk environment with an AWS RDS.

Automated Craigslist Scraper

This is my Craigslist scraper application. Built using Python and using the Selenium automated webdriver framework, this script scrapes listings on craigslist based on type of item, location, max price, etc. This application is deployed/hosted on the Heroku cloud and runs a scrape every hour. It then sends myself a timestamped email with a csv file of the listings that were scraped.

Currently being retrofitted to utilize AWS Lambda, S3, and SNS.
