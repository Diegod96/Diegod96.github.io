Live Twitter Dashboard

Due to the cost of S3 and Databricks, this application is offline. Although, the source code and steps taken to complete this application are in the Github Repository.

This application is a twitter sentiment dashboard. The user can view the live sentiment of twitter along with twitter's sentiment over the current week. The user can also see the live/historic sentiment for a specific topic or term (i.e. COVID, NBA, etc.). There is also a piechart and bar graph displaying the total sentiment of twitter throughout the current week and the top 10 trending hashtags. This application was built using Python, the tweepy API, AWS S3, AWS SQS/SNS, Apache Spark, Databricks, and the Dash - Plotly Python library. Tools & Technologies Used: Python, the tweepy API, AWS S3, AWS SQS/SNS, Apache Spark, Databricks, and the Dash - Plotly Python library

Fake News Detector

This is a fake news detector application utilizing three machine learning models: Logistic Regression, Multinominal Naive Bayes, and SGD Classification. These three models are trained using this dataset of real and fake news. Once the authenticity of the article has been determined, it is displayed to the user via a Flask frontend.

Tools & Technologies Used: Python, Scikit-Learn, Pandas, Flask, and Pickle
